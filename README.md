# Skin Disease Detection System

## Project Overview

The Skin Disease Detection System is a machine learning-based web application that helps identify common skin diseases from uploaded skin images.

## Features

* Upload skin images
* Predict skin disease category
* User-friendly web interface
* Machine Learning based classification

## Technologies Used

* Python
* Flask
* TensorFlow / Keras
* HTML, CSS
* Machine Learning

## Project Structure

* app.py - Main Flask application
* train_model.py - Model training script
* dataset/ - Training and testing images
* templates/ - HTML pages
* static/ - Uploaded images and static files

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Train the model:
   python train_model.py

3. Run the application:
   python app.py

4. Open the browser and visit:
   http://127.0.0.1:5000
